#### 1. Приложение для прослушивания музыки в компании


Веб/мобильное приложение, позволяющее компании друзей или посетителям заведения совместно управлять очередью воспроизведения музыки через голосование.

Процесс включает в себя создание виртуальной «комнаты» для прослушивания музыки компанией, где решение о том, что играть дальше, принимается коллективно.

Ключевое преимущество (killer-фича) заключается в демократизации музыки в офлайн-компании: каждый гость может добавить трек или повлиять на очередь простым голосованием (лайком) со своего телефона, без необходимости устанавливать приложение или иметь платную подписку на Spotify/Apple Music. Хост управляет только колонкой, но не плейлистом единолично.
